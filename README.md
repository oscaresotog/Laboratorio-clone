# Clon de Instagram con Bootstrap

## Descripción del Proyecto

Este proyecto consiste en la recreación visual de la plataforma Instagram utilizando tecnologías de desarrollo web enfocadas en diseño responsivo y componentes reutilizables de Bootstrap.

El objetivo principal fue poner en práctica conocimientos de maquetación web, trabajo colaborativo y personalización de componentes UI mediante HTML, CSS y Bootstrap, simulando la estructura y experiencia visual de una red social moderna.

---

# Tecnologías Utilizadas

- HTML5
- CSS3
- Bootstrap 5

---

# Componentes Bootstrap Implementados

Durante el desarrollo del clon se utilizaron distintos componentes de Bootstrap para lograr una interfaz moderna, adaptable y funcional.

## Componentes utilizados

- Buttons
- Footer
- Carousel
- Grid System
- Forms
- Navbar
- Cards
